# FabLab Website (Previous Version)

The original website for the Gharbiya FabLab, built with basic HTML, CSS, and JavaScript.

## Context & Status

This project is part of my [a8k-legacy](https://github.com/a8k-legacy) archive. It represents an early web development project.

- **Original Purpose:** The first official website for the school's FabLab team, serving as a simple online presence before a more advanced documentation site was built.
- **Key Learnings:** Structuring a multi-page static site, responsive design fundamentals, and deploying a live project.
- **Tech Stack:** HTML, CSS, JavaScript

## Current Version

This website has been succeeded by a new, feature-rich documentation site built with Astro and Starlight.

**Visit the current official website:** [https://stemgharbiya.github.io/fablab/](https://stemgharbiya.github.io/fablab/)  
**View the current project repository:** [github.com/stemgharbiya/fablab](https://github.com/stemgharbiya/fablab)

## Project Note

This repository is preserved in its original state as a record of my early work with the FabLab team. It is not actively maintained.
